# RengeAI Docs

这是一个面向中文场景的 Markdown 文档库，用于系统化描述 RengeAI 接下来要构建的事情、背后的方法论，以及支撑多组织、多项目协作的 AI 智能体团队模型。

## 项目定位

RengeAI 不只是一个单体产品，而是一套围绕 AI 智能体协作展开的组织化工作系统。

它希望解决的问题是：

- 如何让多个 AI 智能体像一个真正的团队一样协同工作
- 如何让团队在多组织、多项目的环境下保持清晰的权限边界
- 如何让需求、研发、交付和复盘形成统一的治理框架
- 如何让项目推进既有工程秩序，也有战略对齐

## 核心设计方向

- 组织结构、项目层级和权限模式，全面参考 GitLab 的体系
- 协作流程包括 GitLab 风格的 Issue、Milestone、MR 等核心机制
- 整体项目管理制度采用 OKR 体系进行目标拆解和推进
- 所有制度设计最终都要服务于 AI 智能体团队的规模化协作
